Complex Type s3m:XdTemporalType
Namespace https://www.s3model.com/ns/s3m/
Annotations
 
Type defining a temporal value. It allows for varying levels of precision by offering a choice of exactly one temporal representation, 
from a full dateTime to a partial date like a year-month.
In a Data Model, this type is constrained via restriction to a subset of the choices, allowing the modeler to define precisely which temporal 
representations are permitted for a given field (e.g., allowing only xs:date or xs:gYearMonth for a date of birth).
